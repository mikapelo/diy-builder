Règles G v2

Moutarde = signature visuelle. Les médaillons d'icônes reviennent en cercle moutarde bordure marine 2px — c'est la marque de fabrique de G et ce qui le rend reconnaissable d'un coup d'œil. Une icône sur trois peut passer en brique pour varier le rythme.

Marine = ossature forte, pas discrète. Border 1.5–2px sur toutes les cards (pas 1px rgba), header de table pleine couleur marine avec texte moutarde, étiquettes en mono uppercase. L'ossature doit se voir.

Brique = CTA primaire + touches rythmiques. Le CTA reste unique, mais la brique revient aussi sur les unités (m², u.) et sur le badge Réf. 24-B pour réchauffer l'ensemble.

Ombre offset douce. Chaque carte porte 0 2px 0 marine + une ombre diffuse 18px — c'est la version moderne du 3px 3px 0 vintage, on garde l'effet « posé » sans l'angle sérigraphique.

Fraunces est remplacé par DM Serif Display pour les titres, chiffres clés et totaux. Plus éditorial, plus affirmé, meilleur ancrage « catalogue français ». Inter pour le corps, IBM Plex Mono pour les réfs et unités.

Ce qui reste banni : halftone en fond plein, box-shadow 3px 3px 0 hard, double border décoratif, arrondis ronds partout. Le G v2 est chaleureux mais pas bricolé.
