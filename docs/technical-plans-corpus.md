# Corpus de référence — Plans techniques construction bois

> Document de recherche EXPORT — DIY Builder
> Date : 2026-03-30
> Objectif : recenser des sources professionnelles réelles de plans techniques pour les 4 modules du simulateur.

---

## Méthodologie de sélection

Chaque source est évaluée selon :
- **Pertinence** : correspond à un module DIY Builder (terrasse, cabanon, pergola, clôture)
- **Qualité technique** : présence de vues normées, cotations, BOM, conventions lisibles
- **Réutilisabilité** : éléments transposables dans les exports PDF du simulateur
- **Fiabilité** : origine professionnelle, guide technique ou norme (pas de contenu décoratif)

Échelle de qualité : ★★★ (référence), ★★☆ (utile), ★☆☆ (partiel)
