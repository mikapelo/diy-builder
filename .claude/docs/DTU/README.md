## ⭐ DTU prioritaires pour DIY Builder

Ces 4 DTU couvrent les 4 modules actifs (cabanon, terrasse, pergola, clôture) :

- **[NF DTU 31.1 — Charpente en bois](./nf-dtu-31-1-p21-203-charpente-en-bois/)** — pergola (poteaux, longerons, chevrons)
- **[NF DTU 31.2 — Construction de maisons et bâtiments à ossature en bois](./nf-dtu-31-2-p21-204-construction-de-maisons-et-batiments-a-ossature-en-bois/)** — cabanon (montants, lisses, contreventement)
- **[NF DTU 41.2 — Revêtements extérieurs en bois](./nf-dtu-41-2-p65-210-revetements-exterieurs-en-bois/)** — cabanon (bardage) + clôture (lames)
- **[NF DTU 51.4 — Platelages extérieurs en bois](./nf-dtu-51-4-p63-205-platelages-exterieurs-en-bois/)** — terrasse (lambourdes, entraxes, fixations)

Utilitaires : **NF DTU 43.4** (toitures bois + étanchéité) pour toit plat cabanon, **FD DTU 31.4** pour façades à ossature bois.
